Style guidelines


<br>                                                <!-- recommended: standalone-tags don't need to be closed                   -->

<br />                                              <!-- however: it is also still allowed to close tags explicitely            -->


<img src="my-image.jpg" />                          <!-- recommended: attribute values should be put in quotes                  -->

<img src=my-image.jpg   />                          <!-- however: unquoted attribute values are also allowed                    -->


<input required />                                  <!-- boolean attributes don't need to have value                            -->

<input required="required" />                       <!-- however: the attribute's name can be used as value                     -->


<em>...</em>                                        <!-- recommended: tags should always be in lowercase                        -->

<eM>...</Em>                                        <!-- however: tags are generally not case-sensitive                         -->



Links


<a href     = "http://google.com.au/"

   download = "save-as-name.txt"                    <!-- force download and (optionally) suggest a save-as filename             -->

   hreflang = "en"                                  <!-- natural language of the linked resource                                -->

   type     = "text/html"                           <!-- the MIME type of the linked resource                                   -->

   media    = "print and (resolution:300dpi)"       <!-- media-query indicating which media the link is optimised for           -->

   target   = "{_blank|_parent|_self|_top|frame}"   <!-- _parent=parent frame, _top=full body of the window                     -->

   rel      = "alternate"                           <!-- alternate presentation of the document (e.g. a translation)            -->

   rel      = "author"                              <!-- link to the document's author                                          -->

   rel      = "bookmark"                            <!-- permanent URL of the current document (used for bookmarking)           -->

   rel      = "external"                            <!-- not part of the current document's website                             -->

   rel      = "help"                                <!-- link to a help document                                                -->

   rel      = "license"                             <!-- link to copyright information                                          -->

   rel      = "next"                                <!-- next document in the series (i.e. next page)                           -->

   rel      = "nofollow"                            <!-- unendorsed document (e.g. advertisement), crawlers should not follow   -->

   rel      = "noreferrer"                          <!-- browser should not send a referrer when following this link            -->

   rel      = "noopener"                            <!-- any browsing context created must not have an opener browsing context  -->

   rel      = "prev"                                <!-- previous document in the series (i.e. previous page)                   -->

   rel      = "search"                              <!-- link to a tool that can be used to search the current document         -->

   rel      = "tag" />                              <!-- a tag/keyword for the current document (probably like a tag cloud)     -->



Inline editing


<div contenteditable="true">                        <!-- make an element editable in the browser (should be activated and       -->

    You can edit this text                          <!-- deactivated through a button outside of the edited container)          -->

</div>


document.designMode = "{on|off}";                   // make the whole doument (or rather only an iFrame) editable
