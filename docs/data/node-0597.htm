Arithmetic operators


+                                           // add

-                                           // subtract

*                                           // multiply

/                                           // division

%                                           // modulo



Comparison operators


==                                          // equal (calls the overloaded operator and checks for value equality)

!=                                          // not equal


eq                                          // reference-equality (pointing to the same object instance)

ne                                          // reference-inequality


<                                           // less than

<=                                          // less than or equal to

>                                           // greater than

>=                                          // greater than or equal to



Logical operators


&&                                          // and

&                                           // and - but always evaluate the right-hand side (no short circuit)

||                                          // or

|                                           // or - but always evaluate the right-hand side (no short circuit)



Bitwise operators


&                                           // bitwise and

|                                           // bitwise or

^                                           // bitwise xor

~                                           // bitwise complement (invert)

<<                                          // shift left

>>                                          // shift right

>>>                                         // unsigned shift right
