XML tags


summary              // short description

remarks              // additional (more extensive) documentation

param                // function parameters

returns              // return value

exception            // exception thrown by the method

typeparam            // describe the types of generics

value                // the value of a property

example              // sampe code demonstrating the usage


c                    // refer to another documented code element

code                 // code snippet

list                 // bulleted or numbered list

para                 // paragraph


paramref             // reference a parameter from the description

see                  // link to another item

seealso              // link to another item
