Defining an interface


Public Interface IPerson                                           ' Interface names always start With an I

    Property FirstName As String                                   ' required properties

    Property LastName As String

    Function FullName() As String                                  ' required methods

End Interface



Implementing an interface


Public Class Employee

    Inherits Object                                                ' inheritance

    Implements IPerson                                             ' interface implementation


    Public Property first As String Implements IPerson.FirstName   ' mark as interface implementation

    Public Property last As String Implements IPerson.LastName     ' the acual property name can differ!


    Public Sub New(first As String, last As String)

        Me.first = first

        Me.last = last

    End Sub


    Function GetFullName() As String Implements IPerson.FullName   ' mark as interface implementation (the method name can differ!)

        Return Me.first & " " & Me.last

    End Function


End Class
