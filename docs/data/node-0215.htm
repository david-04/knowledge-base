Default folders


App_Data                                            // private date (XML, database files, ...)

App_Start                                           // core configuration settings (routes, filters, content bundles, ...)

Areas                                               // partition large applications into smaller units (one subfolder per area)

Content                                             // static assets (CSS style sheets, images, ...)

Controllers                                         // controllers

Models                                              // model classes (domain model, view model, ...)

Scripts                                             // JavaScript libraries

Views                                               // views and partial views

Views/Shared                                        // layouts and views that are not specific to a particular controller
