Defining entities


<!ENTITY dh "David Hofmann">                        <!-- simple text replacement                                          -->


<!ENTITY dh "<firstName>David</firstName>           <!-- can contain nested markup (as a string)                          -->

             <lastName>Hofmann</lastName>">


<!ENTITY footer SYSTEM "footer.xml">                <!-- load from a separate file (add NDATA keyword for non-xml-files=  -->



Using entities in XML


<author>&dh;</author>                               <!-- reference via the &entity; notation                              -->



Using entites in the DTD


<!ENTITY % children "(child1|child2)+">             <!-- defined with the % symbol                                        -->

<!ELEMENT element %children;>                       <!-- referenced with the % symbol                                     -->
