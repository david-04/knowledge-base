Preparing translations


  • install the Multilingual App Toolkit (MAT)
  • set the neutral language for the project (Project Properties -> Application -> Assembly Information)
  • enable localisation via the menu (Tools -> Multilingual App Toolkit -> Enable Selection)
  • right-click on the project in the solution explorer and select Multilingual App Toolkit -> Add Translation Languages
  • rebuild the project and enable "Show all files" in the solution explorer
  • under My Project there are separate files for each culture (Resources.*.resx)
  • for all of these .resx files set the access modifier to Public
  • add the relevant (and to be translated) text content to the .resx files
  • translations are generated in the subfolder MultilingualResources
  • double-click the .xlf file to open the the Multilingual editor and to translate the text content
  • translation files are created in a folder named MultilingualResources
