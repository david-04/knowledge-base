Children


.children()                                         // all direct children

.children(selector)                                 // all direct children matching the given selector

.childAt(0)                                         // the child at the given index



Parents


.parent()                                           // the direct parent

.parents()                                          // all parents (along the whole hierarchy)

.parents(selector)                                  // all parents matching the given selector

.closest(selector)                                  // the closest parent matching the given selector



Search / filter


.find(selector)                                     // all direct and indirect children that match the given selector

.findWhere(wrapper => wrapper.text() !== '123')     // all direct and indirect children for which the given function returns true



Checks


.contains(<div data-test="123">Hello</div>)         // must contain the given element (with exactly the same properties)

.contains([e1, e2...])                              // must contain all of the given elements


.containsMatchingElement(<a data-test="1">Hi</a>)   // must contain the given element (with at least the given properties)

.containsAllMatchingElements([e1, e2...])           // must contain all of the given elements

.containsAnyMatchingElements([e1, e2...])           // must contain at least one of the given elements
