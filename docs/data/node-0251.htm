Setting the culture


<%@ Page ... UICulture="en-AU" Culture="en-AU" %>                  <!-- set the page to a fixed culture                          -->


<%@ Page ... UICulture="auto" Culture="auto" %>                    <!-- autoselect the culture based on the client's preferences -->


<%@ Page ... UICulture="auto:en-AU" Culture="auto:en-AU" %>        <!-- autoselect if supported, use fixed culture as fallback   -->


<configuration>

  <system.web>

    <globalization culture="en-AU" uiCulture="en-AU" />            <!-- set the culture for all pages                            -->


Thread.CurrentThread.CurrentCulture = new CultureInfo("en-AU");    // set the culture programatically



Local resources


Design view -> Tools -> Generate Local Resource


<asp:TextBox ... meta:resourcekey="TextBox1Resource1">             <!-- meta:resourcekey replaces the Text attribute             -->


TextBox1Resource1.Text                                             // a form-specific resource file is generated automatically in

TextBox1Resource1.ToolTip                                          // ./App_LocalResources/WebFormName.aspx.resx


GetLocalResourceObject("TextBox1Resource1.Text");                  // local resources can be retrieved programatically



Global resources


Must be placed in folder /App_GlobalResources


Resources.Resources.Greeting                                       // access global resources (Resources.ResourceFileName.Property)


<asp:Literal ... Text="<%$ Resources: Resources, Greeting %>" />   <!-- reference global resources from the web form             -->
