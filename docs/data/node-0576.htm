Command line


*.scss                                              /* new syntax (Sassy CSSS) using {} and ; for grouping/delimiting       */

*.sass                                              /* old syntax, using ^\t and \n instead of {} and ;                     */


> sass-convert my-file.scss my-file.sass            /* convert between the the two syntaxes (works in both directions)      */


> sass input.scss output.css                        /* compile a Sass input file into standard CSS output                   */


> sass --sourcemap=none --watch in.scss:out.css     /* compile in.scss -> out.css whenever the file changes (watchdog)      */

> sass --watch inFolder:outFolder                   /* watch and recompile all files form the inFolder to the outFolder     */


> sass -i                                           /* run an interactive shell to experiment with SassScript               */


> sass --style=nested                               /* indent rules if they are logical children of the preceding rule      */

> sass --style=expanded                             /* do not indent rules (only properties within them)                    */

> sass --style=compact                              /* one line per rule (selector and properties are all in one line)      */

> sass --style=compressed                           /* remove all unnecessary white space                                   */

Namespaced properties


#myElement {

    margin: {                                       /* works like "WITH" keyword in VB, compiles to:                        */

        left:  1em;                                 /* margin-left:  1em;                                                   */

        right: 2em;                                 /* margin-right: 2em;                                                   */

    }

}



Including files


@import "file-1.scss", "file-2.scss";               /* like C's #include (insert the specified files here)                  */


#myElement {

    @import "specific-styles.scss";                 /* the import can be limited to the current element's scope             */

}



Debugging


@debug My variable is #{$myVariable}                /* print a debug message while compiling                                */

@warn  My variable is #{$myVariable}                /* print a warning (can be deactivated) along with a stack trace        */

@error My variable is #{$myVariable}                 /* throw an error and abort the compilation process                    */
