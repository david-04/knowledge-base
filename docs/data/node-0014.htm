Discovering collections


show collections                                    // list all collections in the current database (interactive shell only)

db.getCollectionNames()                             // this syntax must be used from within scripts


db.mycollection                                     // the collections are accessible as properties of the db object

db.getCollection("mycollection")                    // alternative syntax (useful if the collection name is a reserved word)

db["mycollection"]                                  // same as above


db.mycollection.stats()                             // show statistics regarding the specified collection



Creating collections


db.createCollection("mycollection");                // create a collection (db + collection name should be <= 121 (better: <100) bytes)


db.createCollection("mycollection", {               // capped collections have a fixed size implementing a circular buffer storage

    "capped" : true,                                // when the collection is full, new documents replace old ones (documents can't be

    "size"   : 100000 /* bytes */,                  // deleted manually and updates are allowed only if the document size does not change)

    "max"    : 100 /* documents */                  // optional: keeps at most "max" documents (or less if the "size" limit is hit first)

});



Altering collections


db.mycollection.renameCollection("mycollection2")   // rename a collection

db.mycollection.renameCollection("mycoll2", true)   // rename a collection and replace "mycoll2" if it already exists


db.runCommand({                                     // convert a regular collection into a capped one

    "convertToCapped" : "mycollection",

    "size"            : 10000

});


db.mycollection.drop()                              // delete a collection

db.runCommand({"drop" : "mycollection"})            // same as above (different syntax)


db.runCommand({                                     // create a copy of an existing collection

    "cloneCollection" : "mycollection",             // ... with the given name

    "from"            : "hostname:27017"            // ... that currently resides on the specified mongod

})



Data types


null                                                // empty or non-existent field

true / false                                        // boolean values

3.5                                                 // 64-bit floating point numbers

NumberInt("3")                                      // 4-byte signed integer: -2,147,483,648 ... +2,147,483,647

NumberLong("3")                                     // 8-byte signed integer: -9,223,372,036,854,775,808 ... 9,223,372,036,854,775,807

"string"                                            // UTF-8 encoded strings

new Date() / 1482355706                             // dates are stored as milliseconds since 1970 (call new Date(), not Date() [which returns a string])

/regexp?/                                           // regular expressions are used in queryies only

[1, "two", 3]                                       // arrays can contain mixed content

_id = ObjectId()                                    // 12-byte unique ID for each document (generated from timestamp+machine+pid+increment)

function(){...}                                     // functions
