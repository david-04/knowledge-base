System databases


admin                                               // the root database (in terms of authentication)

local                                               // data that's local to the server instance (will never be replicated)

config                                              // stores information about a shards in a sharded setup



Database commands


db                                                  // represents the currently connected database


connection = new Mongo("my-host:30000")             // connect to another MongoDB server

db = connection.getDB("myDB")                       // swith to a database on the other server


show dbs                                            // list all databases (interactive shell only)

db.getMongo().getDBs()                              // this syntax must be used from within scripts


use mydatabase                                      // switch to another database (interactive shell only)

db.getSisterDB("mydatabase")                        // this syntax must be used from within scripts
