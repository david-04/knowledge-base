Listing tags


> git tag                                           // show a list of all tags

> git tag -l "my*Tag"                               // list all tags matching the given filter

> git show myTag                                    // show information regarding the specified tag



Creating tags


> git tag    myTag                                  // add a simple tag (without any metadata) to the most recent commit

> git tag -a myTag                                  // add an annotated tag (with SHA-1, name, email, ...)

> git tag -s myTag                                  // add a signed tag


> git tag -a myTag 9fceb02                          // add a tag to the commit with the specified SHA-1



Using tags


> git checkout -b myBranch myTag                    // create and checkout a new branch based on the given tag
