Compacting


db.runCommand({ "compact": "collName" })


db.runCommand({ "compact"       : "collName",       // defragment the database

                "paddingFactor" : 1.5 })            // optional: add padding to each document (defaut: 1=no padding)


db.repairDatabase()                                 // recreate (compact+defragent+shrink) a database (to reclaim unused disk space)


> mongod --repair                                   // recreate all databases maintained by a server




Assorted


db.adminCommand({ "setParameter"          : 1,     

                  "journalCommitInterval" : 10 })   // persist the journal every 10ms (default: 100ms, can be set to 2ms...500ms)


db.mycollection.validate()                          // check a collection for corruptions


db.adminCommand({"setParameter" : 1,                // set the loglevel

                 "logLevel"     : 3})               // 0 (default, little logging) ... 5 (verbose)


db.startup_log                                      // contains one summary document for each server startup
