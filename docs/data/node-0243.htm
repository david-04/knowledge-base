Container and grouping


PlaceHolder        => group multiple elements without adding any encapsulating HTML element

Panel                => group multiple elements represented as a DIV in the final HTML output

Wizard        => splitting page into mulitple forms that the user fills in one step at a time (navigation between the steps is added automatically)

MultiView        => like Wizard but navigation must be handled manually/programmatically



Using the wizzard


<asp:Wizard ID="Wizard1" runat="server">

    <WizardSteps>

        <asp:WizardStep ID="Step1" runat="server" Title="Name">

            Name: <asp:TextBox ID="TextBox1" runat="server"></asp:TextBox>

        </asp:WizardStep>

        <asp:WizardStep ID="Step2" runat="server" Title="Age" StepType="Finish">

            Age: <asp:TextBox ID="TextBox2" runat="server"></asp:TextBox>

        </asp:WizardStep>

        <asp:WizardStep ID="Step3" runat="server" StepType="Complete">

            Thank you!

        </asp:WizardStep>

    </WizardSteps>

</asp:Wizard>
