Including scripts


@include "test2"



Line continuation


,  ?    :    ||    &&                       # after any of those the command can be continue on the next line without preceding \



Dynamic function call


callMe = "myFunction"                       # the function name must be assigned to a variable

result = @callMe()                          # call the function throug the name stored in the variable



Switch statement


switch(variable) {                          # the switch-statement is an AWK extension


    case "string":                          # value comparison with numbers or strings (case sensitive)

        break


    case /[a-z]/:                           # regular expressions can be used as well (an match only a part of the expression)

        break


    default:

}



Debugging


> gawk --debug -f program.awk input.dat     # start gawk in debugging mode


r                                           # run to the next breakpoint

bt                                          # show the current stacktrace (backtrace)

p my_variable1 my_variable2                 # print variable values

p @my_array                                 # print all array elements

n                                           # proceed to the next command

q                                           # quit the debugger


[Return]                                    # repeat the previous command



Predefinied variables


ENVIRON["ENVIRONMENT_VARIABLE"]             # contains the system's environment variables


FUNCTAB                                     # array with all built-in and user-defined function names as index elements

SYMTAB                                      # array with all global variables and arrays (assignable: SYMTAB["my_variable"] = 1)


ERRNO                                       # error message after getline or close failed (checked in BEGINFILE to spot unreadable files)

LINT=1                                      # show Lint warnings

BINMODE=3                                   # set this to 3 to use binary mode for all file I/O

ARGC + ARGV                                 # contains only awk.exe + input files + assignments [var=value without -v] (can be modified as well)

ARGIND                                      # ARGC index pointing to the currently processed input file


PROCINFO["identifiers"][my_variable]        # contains the type of the object (array, extension, scalar, untyped, [functions:] builtin, user)

PROCINFO["pgrpid"]                          # process group ID

PROCINFO["pid"]                             # process ID

PROCINFO["ppid"]                            # parent process ID

PROCINFO["strftime"]                        # default time format string for strftime()

PROCINFO["uid"]                             # user ID

PROCINFO["input_name","READ_TIMEOUT"]=1000  # set the read timeout to 1000 ms (input name can be "/inet/tcp/0/localhost/80" or "/dev/stdin")

PROCINFO["sorted_in"]                       # defines the sort order when traversing arrays via for-in loops
