Layout managers (AWT)


component.setLayout(null);                          // disable the layout manager (and position everything manually)

component.setLayout(new GridLayout());              // set a specific layout manager



Layout managers (AWT)


FlowLayout                                          // default layout manager: place one item after the other and break lines

BorderLayout                                        // north, south, east, west, center

GridLayout                                          // like a table with rows and columns, content is placed in single cells

GridBagLayout                                       // similar to GridLayout, but cells can have different sizes and each row

                                                    // can have a different number of columns

CardLayout                                          // similar to tabs, but the tabs themselves are not displayed, switching

                                                    // between the tabs can be accomplished only programatically
