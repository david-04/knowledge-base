Number formatting


format-number(123)                                  // format a numeric value

format-number(123, "#,##0.###")                     // optionally apply a format string:

                                                    // 0 ... a digit including leading and trailing zeros

                                                    // # ... a digit but no leading or trailing zeros

                                                    // . ... decimal separator

                                                    // , ... grouping seprator (groups of thousands)

                                                    // ; ... positive and negative subformats (e.g. ##0.##; -##0.##)

                                                    // - ... negative prefix

                                                    // % ... multiply by 100 and show with a percent sign

                                                    // %. ... multiply by 1,000 and show with per mille sign

                                                    // X ... any other character can be used in the prefix or suffix

                                                    // ' ... quote literal text
