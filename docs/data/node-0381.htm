Physical markup


<span>                                              <!-- empty inline-wrapper                                | format: default      -->

<b>                                                 <!-- bold                                                | format: bold         -->

<i>                                                 <!-- italic                                              | format: italic       -->

<u>                                                 <!-- underlined                                          | format: underlined   -->

<s>                                                 <!-- striked out                                         | format: striked out  -->

<mark>                                              <!-- highlighted with yellow background                  | format: highlighted  -->


<sub>                                               <!-- subscript                                           | format: subscript         -->

<sup>                                               <!-- superscript                                         | format: superscript       -->

<small>                                             <!-- small font size                                     | format: small         -->


<br>                                                <!-- line break (clear="all" is no longer supported)     |                      -->

<hr>                                                <!-- horizontal rule                                     | format: ----------   -->



Semtantic formatting


<strong>                                            <!-- marked passage has an important meaning             | format: bold)        -->

<em>                                                <!-- marked passage is pronounced with empathetic stress | format: italic       -->

<s>                                                 <!-- the content is no longer correct                    | format: striked out  -->


<del                                                <!-- text that was deleted                               | format: striked out  -->

     cite     = "http://go.to/312"                  <!-- URL explaining the reaon for deletion               |                      -->

     datetime = "2016-12-24T18:45:33Z">             <!-- delete date/time (Z=time zone designator for GMT)   |                      -->


<ins                                                <!-- text that was inserted                              | format: underlined   -->

     cite     = "http://go.to/312"                  <!-- URL explaining the reaon for insertion              |                      -->

     datetime = "2016-12-24T18:45:33Z">             <!-- insert date/time (Z=time zone designator for GMT)   |                      -->



Code related markup


<code>                                              <!-- source code                                         | format: monospace    -->

<var>                                               <!-- variable                                            | format: italic       -->

<kbd>                                               <!-- keyboard input                                      | format: monospace    -->

<samp>                                              <!-- sample program output                               | format: monospace    -->

<output form="myFormId" for="field1Name field2..."> <!-- data generated from the given form's fields         |                      -->



Citation and quotes


<q>                                                 <!-- inline quote                                        | format: "in quotes"  -->

<cite>                                              <!-- inline quote                                        | format: italic       -->

<blockquote cite="http://go.to/2342">               <!-- a quoted section (optional: from the given source)  | format:     indented -->



Abbreviations

<dfn>HTML</dfn> is the standard markup ...          <!-- first mentioning of a term + its definition         | format: italic       -->
