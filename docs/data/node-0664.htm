Dates


Dim Now As DateTime = DateTime.Now                    ' get the current time


Dim past As DateTime = New DateTime(2015, 12, 24)     ' Set a predefined time

past = #2015/12/24 23:00:00#                          ' set the date as a constant

past = Date.Parse("2015/12/26 23:00:00")              ' also avalable as TryParse


Console.WriteLine((past - Now).Days)                  ' calculate time differences

Console.WriteLine(Now.ToString("yyyy/MM/dd"))         ' format as specified

Handling overflows


  • overflows raise exceptions by default and can't be disabled
  • to discard overflow exceptions calculate with a larger type (like Long) and apply "Mod Integer.MaxValue"
