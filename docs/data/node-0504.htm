Node package manager


> npm {install|update|uninstall} 'sax'              // install/update/delete the package (latest version)


> npm {install} 'sax@0.3.x'                         // install a specific package version


> npm {install|update}                              // install all packages listed in package.json (see below)


> npm -g {install|update|uninstall} 'sax'           // install/update/delete global packages (in/into/from the Node installation directory)


> npm install --save 'sax'                          // install the package and add it to the package.json


> npm install --save-dev 'sax'                      // install the package and add it to the development section of package.json



Configuring package.json


{                                                   // package.json must be placed in the application's root folder

    'name'    : 'MyApp',

    'version' : '1.0.0',

    'dependencies' : {                              // define package dependencies (these are by default only for the DEV environment)

        'nano'      : '*',                          // use the latest version

        'sax'       : '0.3.x',                      // use any version that starts with 0.3.

        'request'   : '>0.2.0',                     // use any version above 0.2.0

        'express'   : '^4.0.0',                     // use any version in [4.0.0; 5.0.0[

        'underscore': '>=0.1.0 <0.3.4'              // use any version between 0.1.0 and 0.3.3

    }

}


> npm init                                          // auto-create an initial package.json
