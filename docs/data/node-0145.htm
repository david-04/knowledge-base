> java -classpath ./libs/jooq-3.9.5.jar;./libs/jooq-codegen-3.9.5.jar;./libs/jooq-meta-3.9.5.jar;./libs/mysql-connector-java-5.1.43-bin.jar;. org.jooq.util.GenerationTool my-jooq-config.xml


or more generic:


> java -classpath `find . -name "*.jar" | sed -z -E "s/\s+/;/g;s/$/;./g"` org.jooq.util.GenerationTool jooq-codegen.xml
