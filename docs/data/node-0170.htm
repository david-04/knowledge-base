Setup


> npm install create-react-app                      // install the build tools

> create-react-app my-app                           // create the folder "my-app" and initialise it with a sample application


my-app> npm start                                   // start the development server and the automated build tools

my-app> npm run build                               // or create a pre-compiled (optimised) version in the build folder





> npm install react react-dom                       // install the React libraries only (without build tools and test server)


<script src="node_modules/react/dist/react.js"></script>

<script src="node_modules/react-dom/dist/react-dom.js"></script>
