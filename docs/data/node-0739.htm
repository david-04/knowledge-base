Command line parameters


gradle -q ...                               // run in silent mode


gradle myTask                               // run a single task (including all dependencies)

gradle myTask -x myOtherTAsk                // run a task but without resolving the specified dependency


gradle myTask -continue                     // don't stop on errors, continue building/compiling

gradle -b another.gradle myOtherTask        // execute another build file


gradle projects                             // list all projects

gradle tasks                                // list all tasks (seems to look into projects/modules only)
