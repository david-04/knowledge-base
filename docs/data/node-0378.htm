Basic string functions


number = strtonum("1.52")                         # parse a numeric string into a numeric value

number = strtonum("0xFF")                         # supports octal (0...) and hexadecimal (0x...) notation


string = sprintf("%s ...", var, [...])            # format a string


length = length("Hello world!")                   # determine the length of a string (counts characters, not bytes)


string = substr(string, start, [length])          # extract a substring from the given start index


string = tolower(sring)                           # to lower case

string = toupper(string)                          # to upper case



Search and replace


position = index("Hello world!", "world")         # returns the index (1-based) or 0 if no match was found


position = match("text", /regexp/, [array])       # find /regexp/ in "text" and optionally store the result in array, returns the start index of the match

                                                  # array[0] == the whole match

                                                  # array[1] == sub-match that belongs to the first grouping bracket

                                                  # array[1,"start"]  == start index of the first sub-match

                                                  # array[1,"length"] == length of the first sub-match

sub(/regexp/, "replace", [srcTarget])             # replace the first /regexp/ match by "replace" in srcTarget (or $0, if the parameter is missing)

                                                  # "&" in the replacement text is subsituted by the current match


count = gsub(/regexp/, "replace", [srcTarget])    # replace all matches and return the number of replacements made


string = gensub(/regexp/, "replace", "g", [src])  # replace all /regexp/ by "replace" in src but do not overwrite src but return the result

string = gensub(/regexp/, "replace", 5, [src])    # only replace the 5th match

string = gensub(/regexp/, "\\1", "g", [src])      # the replacement can contain \\1...\\9 to reference submatches



Splitting


count = split("text", array, [fieldsep, seps])    # split the text into array[1...count] using the fieldsep (default: FS)

                                                  # the separators are stored in seps[0...count]


count = patsplit("text", array, [fieldpat, seps]) # split the text into array[1...count] using the field pattern (default: FPAT)

Regular expressions


{5}      # exactly 5 times

{5,}     # at least 5 times

{5,10}   # 5 ... 10 times


\s       # whitespace

\S       # non-whitespace

\w       # word (letter, digit, underscore)

\W       # non-word

\<       # beginning of word

\>       # end of word


\B       # must have word (letter, digit, underscore) but is not considered part of the match



Format string


%2$d %1$s ................... # "abc", 123    => "123 abc"


%s .......................... # "Hello world" => "Hello world"

%5s ......................... # "Hello world" => "Hello world"

%-5s ........................ # "Hello world" => "Hello world"

%15s ........................ # "Hello world" => "     Hello world"

%-15s ....................... # "Hello world" => "Hello world     "


%.5s ........................ # "Hello world" => "Hello"

%10.5s ...................... # "Hello world" => "     Hello"

%-10.5s...................... # "Hello world" => "Hello     "


%c .......................... #  65           => "A"


%x .......................... # 255           => ff

%X .......................... # 255           => FF

%#X ......................... # 255           => 0xFF


%d .......................... #   3.9         => "3"

%3d ......................... #   5           => "  5"

%-3d ........................ #   5           => "5  "

%.3d ........................ #   5           => "005"

%5.3d ....................... #   5           => "  005"

%-5.3d ...................... #   5           => "005  "

%+...d ...................... #  ¬±5           => "¬±5"


%e .......................... #  12.3e6       => "1.230000e+07"

%E .......................... #  12.3e6       => "1.230000E+07"


%f .......................... #  12.3456      => "12.345600"

%12f ........................ #  12.3456      => "   12.345600"

%-12f ....................... #  12.3456      => "12.345600   "

%012f ....................... #  12.3456      => "00012.345600"

%.3f ........................ #  12.3456      => "12.346"

%10.3f ...................... #  12.3456      => "    12.346"

%-10.3f ..................... #  12.3456      => "12.346    "

%+...f ...................... # ¬±12.3456      => "¬±12.345600"


%g .......................... # select %f or %F depending on the number's size
