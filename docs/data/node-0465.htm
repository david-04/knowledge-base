Inline styles


element.style.backgroundColor = "white"             // L0: access single style properties

element.style.cssFloat = "right"                    // L0: float is reserved name in JS, so the property is called cssFloat

element.style.width = "10px"                        // L0: all measurements need to have a unit


element.style.cssText                               // L2: get (over overwrite) all CSS properties as a single string


element.style.length                                // L2: number of properties set

element.style.item[0]                               // L2: returns the name of the CSS property at the given position

element.style.item(0)                               // L2: returns the name of the CSS property at the given position


element.style.getPropertyValue("height")            // L2: get the property's value as string

element.style.getPropertyCSSValue("width")          // L2: retrieve a property as a CSSValue object

element.style.getPropertyPriority("border")         // L2: returns "important" or ""

element.style.removeProperty("color")               // L2: remove the given style

element.style.setProperty("top", "0", "important")  // L2: set a property with the given priority ("important" or "")


element.style.parentRule



Computed styles


style = window.getComputedStyle(element, ":after")  // L2 (except IE): retrieve all applied style properties (second parameter can be "")

style = element.currentStyle                        // L2 (IE only):   same of the above


style.backgroundColor                               // L2: some browsers return "rgb(255,0,0)" or "#ff0000" instead of "red"

style.border                                        // L2: this is often not returned (only borderTopStyle, borderBottomColor, ...)



Styles sheets (embedded or external)


document.styleSheets[0]                             // L2: collection of all style sheets associated with the document


styleSheet.disabled                                 // L2: en- or disable a single style sheet

styleSheet.href                                     // L2: the style sheet's URL or null (if embedded)

styleSheet.media                                    // L2: collection (like an array) of supported media types

styleSheet.ownerNode                                // L2: link to the <link> or <style> element

styleSheet.parentStyleSheet                         // L2: set if the stylesheet was imported by another one (via @import)

styleSheet.title                                    // L2: the value of the title attribute on the ownerNode

styleSheet.type                                     // L2: usually "text/css"


styleSheet.cssRules                                 // L2: collection of all CSS rules contained in the stylesheet (older IE versions: styleSheet.rules)

styleSheet.ownerRule                                // L2: the owner's rules (for imported style sheets, otherwise null)

styleSheet.deleteRule(index)                        // L2: delete the rule with the given index (older IE versions: styleSheet.removeRule)

styleSheet.insertRule("p{color:red;}", index)       // L2: insert the rule before the given index (older IE versions: styleSheet.addRule("p","color:red;",index))


cssRule.cssText                                     // L2: the whole rule as a string

cssRule.parentRule                                  // L2: the @import rule (in case of imported style sheets, othwerwise null)

cssRule.parentStyleSheet                            // L2: the style sheet this rule belongs to (not supported by IE)

cssRule.selectorText                                // L2: the selector text (often read-only)

cssRule.style                                       // L2: the style object to get/set the individual properties

cssRule.type                                        // L2: always 1 for style rules (not supported by IE)
