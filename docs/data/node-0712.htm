Importing sub-schemas


<xs:include schemaLocation="my-sub-schema.xsd"/>    <!-- include an XSD defined in the same namespace as the current file  -->



Overriding sub-schema elements


<xs:redefine schemaLocation="my-sub-schema.xsd">    <!-- include this XSD but override some of its definitions             -->


  <xs:complexType name="myType">

    <xs:complexContent>

      <xs:extension base="xs:string">

        <xs:attribute name="version" type="xs:int"/>

      </xs:extension>

    </xs:complexContent>

  </xs:complexType>


</xs:redefine>



Importing other namespaces


<xs:import

    namespace="http://my-domain.com/MyNamespace"    <!-- import all global types and elements into the current XSD         -->

    schemaLocation="./my-schema.xsd"/>
