Exiration


EXPIRE          myKey 60                            // expire the given key in 60 seconds

EXPIREAT        myKey 1435717600                    // expire the given key at the given timestamp

PERSIST         myKey                               // remove the expiration timeout from the key

TTL             myKey                               // show the remaining life-span of the given key in seconds

PTTL            myKey                               // show the remaining life-span of the given key in milliseconds

                                                    // -1 ... will never expire (no expiration date/time set)

                                                    // -2 ... has already expired
