Loosely coupled file resources


Add the files to the solution and set the following properties:


BuildAction           = Content

CopyToOutputDirectory = Always


<Image x:Name="ImageHolder" Stretch="Fill" />


this.ImageHolder.Source = new BitmapImage(new Uri($@"{Environment.CurrentDirectory}\Images\flag.jpg"));


The resources (including subdirectories) are copied to the Bin folder and loaded from the file system at runtime



Embedded file resources


Add the files to the solution and set the following properties:


BuildAction           = Resource

CopyToOutputDirectory = DoNotCopy


BitmapImage image = new BitmapImage(new Uri(@"/Images/flag.jpg", UriKind.Relative));


The resources (with their relative paths) are compiled into the assembly and retrieved from there at runtime



Logical window resources


<Window ...>

    <Window.Resources>

        <LinearGradientBrush x:Key="MyBrush" EndPoint="0.5,1" StartPoint="0.5,0">

            <GradientStop Color="#4CFF0000" Offset="0"/>

            <GradientStop Color="White" Offset="1"/>

        </LinearGradientBrush>



Logical application resources


<Application ...>

    <Application.Resources>

       <SolidColorBrush x:Key="MyBrush" Color="blue"/>



Logical reuseable resources


Create a XAML Resource Dictionary file:


<ResourceDictionary ...>

    <Style x:Key="MyStyle">

        <Setter Property="Control.FontSize" Value="20"/>

    </Style>

</ResourceDictionary>


Reference the XAML Resource Dictionary from App.xaml


<Application ...>

    <Application.Resources>

        <ResourceDictionary>

            <ResourceDictionary.MergedDictionaries>

                <ResourceDictionary Source="MyStyles.xaml"/>

            </ResourceDictionary.MergedDictionaries>

        </ResourceDictionary>

    </Application.Resources>

</Application>


For style files in external libaries/assemblies add the library name and the Component keyword


<ResourceDictionary.MergedDictionaries>

    <ResourceDictionary Source="/MyResourcesLibrary;Component/MyBrushes.xaml"/>

</ResourceDictionary.MergedDictionaries>
