Authentication


requirepass myPassword                              // in redis.conf: set a password (should be at least 64 bytes long)


> redis-server /home/~david/redis.conf              // start the server with a given configuration file


AUTH myPassword                                     // the client sends the password unencrypted to the server



Renaming commands


                                                    // create a file "rename-commands.conf" to rename built-in commands:

rename-command FLUSHDB e0cc96ad2eab73c2c34701180    // the FLUSHDB command is now called e0cc96ad2eab73c2c34701180

rename-command FLUSHALL a31907b21c437f46808ea4932

rename-command CONFIG ""                            // the CONFIG command is no longer accessible at all

rename-command KEYS ""

rename-command DEBUG ""

rename-command SAVE ""


include /home/~david/rename-commands.conf           // include the rename-commands.conf from within redis.conf



Block external access


bind 127.0.0.1                                      // in redis.conf: listen only on the loopback interface
