General Annotations


@NonNull                                    // passing in null as a parameter will throw an exception straight away


@Cleanup                                    // automatically call myInputStream.close() at the end of the scope


@SneakyThrows(IOException.class)            // the method throws this (or any if not specified) exception without declaring it


@Synchronized                               // mark a method as synchronised (locks a private property $lock instead of this)
