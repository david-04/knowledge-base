Creating stashes


> git stash save                                    // move all staged + unstaged modifications to a backup stack

> git stash save --keep-index                       // stash unstaged modifications only (preservere staged modifictions)

> git stash save --include-untracked                // stash untracked files as well (by default, only tracked files are stashed)

> git stash save --patch                            // interactively select the components to stash



Analysing stashes


> git stash list                                    // show all stashed states



Restoring stashes


> git stash apply stash@{2}                         // restore the latest (or the specified) stash snapshot to the currently checked

                                                    // out version (this can be another branch than the one from which the stash was

                                                    // originally created)


> git stash apply stash@{2} --index                 // restore staged files as well (by default, they are skipped)


> git stash branch myBranch                         // create myBranch based on the stash's original branch and apply the stash data


> git stash pop stash@{2}                           // restore the most recent (or specified) stash and remove it from the stash afterwards


> git stash drop stash@{2}                          // delete the most recent (or the specified) stash
