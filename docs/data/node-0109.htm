Screen sizes

sm ... small           [768px,992px[                <!--                   750px                                          -->

md ... medium                [992px,1200px[         <!--                   970px                                          -->

lg ... large                       [1200px,oo[      <!--                  1170px                                          -->

Grid system

    <div class="row">                               <!-- a row can actually wrap its cells across multiple rows           -->

        <div class="col-sm-6 col-md-4 col-lg-3">    <!-- this cell occupies 6/12 columns on small screens an 3/12 on      -->

            Content                                 <!-- large ones (xs is set to 12/12 [one column] by default)          -->

        </div>

        <div class="col-sm-6 col-md-4         ">    <!-- md is applied to large screens if lg is missing                  -->

            Content

        </div>

        <div class="clearfix visible-sm-block">     <!-- on small screens the row will break here (6+6=12colums)          -->

        </div>                                      <!-- add a clearfix that is only visible on small screens             -->

        <div class="col-sm-6 col-md-4 col-lg-3">

            Content

        </div>

        <div class="clearfix visible-md-block">     <!-- on medium screens the row will break here (4+4+4=12colums)       -->

        </div>                                      <!-- add a clearfix that is only visible on small screens             -->

        <div class="col-sm-6 col-md-4 col-lg-3">

            <div class="row">                       <!-- grids can be nested                                              -->

                <div class="col-sm-6 col-md-4">     <!-- there are 12 columns to fill (like for top-level grids)          -->

                </div>

Grid classes


row                                                 <!-- a single row (can actually spread accross multiple rows)         -->


col-{xs|sm|md|lg}-{1|...|12}                        <!-- cell occupying x columns on the specified screens size           -->


col-{xs|sm|md|lg}-push-{1|...|11}                   <!-- move the cell to the right (usefule for swapping cells on        -->

col-{xs|sm|md|lg}-pull-{1|...|11}                   <!-- different screen sizes)                                          -->



col-{xs|sm|md|lg}-offset-{1|...|12}                 <!-- move the cell x columns to the right (adds margin-left)          -->

col-{   sm|md|lg}-offset-0                          <!-- clear the offset (required for the next-larger screen size)      -->


clearfix                                            <!-- clear floats on both sides (after the element)                   -->


visible{xs|sm|md|lg}-{inline|block|inline-block}    <!-- visible only for the selected screen size(s)                     -->

hidden-{xs|sm|md|lg}                                <!-- hidden only for the selected screen size(s)                      -->
