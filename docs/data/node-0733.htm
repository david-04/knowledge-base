Key management


> gpg --gen-key                                     // generate a GPG key pair

> gpg --list-keys                                   // list all GPG keys

> git config --global user.signingkey 0A46826A      // register the ID of the private key (as shown by gpg --list-keys)


> gpg -a --export 0A46826A                          // export the public key

            | git hash-object -w --stdin            // and import it as a BLOB into the Git repository

> git tag -a davids-pgp-pub 659ef797d18...          // create a tag (pass the above command's SHA-1 as parameter)

> git push --tags                                   // push the public key and its tag into the origin repository


> git show davids-pgp-pub | gpg --import            // import a pbulic key from the Git repository into the GPG key ring



Signing and verification


> git tag -s myTag                                  // create a signed tag

> git tag -v myTag                                  // verify the signature of a tag


> git commit -S                                     // create a signed commit

> git log --show-signature                          // verify the signatures of signed commits


> git merge --verify-signatures myBranch            // merge only if all affected commits are signed and can be verified

> git merge -S myBranch                             // sign the result of a merge (i.e. the resulting commit)
