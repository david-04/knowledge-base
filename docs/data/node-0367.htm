Media types


<link href  = "my-styles.css"                       <!-- include an external style sheet                                         -->

      rel   = "stylesheet"                          <!-- type of the link must be "stylesheet"                                   -->

      type  = "text/css"                            <!-- optional: the MIME type (can safely be emitted for rel="stylesheet")    -->

      media = " {all|print|screen|speech}"          <!-- restrict to a selected media type (or "all")                            -->

      media = "screen and (max-width: 568px)" />    <!-- ... or use media queries (not supported in IE<=8; use fallback below)   -->

<style type  = "text/css"                           <!-- inline style definitions (the MIME type is optional and can be omitted) -->

       media = " {all|print|screen|speech}"         <!-- restrict to a selected media type (or "all")                            -->

       media = "screen and (max-width: 568px)" />   <!-- ... or use media queries (not supported in IE<=8; use fallback below)   -->


@media {all|print|screen|speech|media-query} {...}  /* make some rules within a CSS stylesheet media-specific                     */



Media queries


@media        screen and (color) {...}              /* media queries are always in parentheses (ignored by older browsers)        */

@media only   screen and (color) {...}              /* prevent older browsers from (falsely) interpreting the query               */

@media not    screen and (color) {...}              /* not (screen and color)                                                     */

@media print, screen and (color) {...}              /* logical OR                                                                 */



{min|max}-width:              1280px                /* viewport width (i.e. width of the browser window)                          */

{min|max}-height:             720px                 /* viewport height (i.e. height of the browser window)                        */


{min|max}-device-width:       1280px                /* physical screen resolution                                                 */

{min|max}-device-height:      720px                 /* physical screen resolution                                                 */


          orientation:        {potrait|landscape}   /* screen orientation (base on viewport's width/height ratio)                 */


{min|max}-aspect-ratio:       16/9                  /* width/height ratio (based on the viewport)                                 */

{min|max}-device-spect-ratio: 1.7778                /* width/height ratio (based on the physical screen size)                     */


{min|max}-resolution:         200dpi                /* physical resolution of the display                                         */


{min|max}-color:              8                     /* colour depth in bits                                                       */



Useful queries


max-width:568px                                     /* iPad (portrait or landscape)                                               */

max-width:768px and orientation:potrait             /* iPad portrait                                                              */

max-width:768px and orientation:landscape           /* iPad landscape                                                             */
