Defining variables


<xsl:variable name   = "myVariable"                 <!-- define a variable                                                 -->

              select = "@myAttribute" >             <!-- initialise it with this XPath result                              -->

Using variables


<xsl:value-of select = "$myVariable" />             <!-- access the variable's current value through an XPath expression   -->
