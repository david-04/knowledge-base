Defining structures


<StructLayout(LayoutKind.Auto)>                         ' fields and properties are aligned/reordered for best performance

Public Structure MyDate                                 ' a structure is the value-type equivalent of a class


    Public Property Day As Integer                      ' it contains fields (or automatic properties)

    Public Property Month As Integer

    Public Property Year As Integer


    Public Sub New(Day As Integer,                      ' the default constructor can't be overloaded,

                   Month As Integer,                    ' only constructors with parameters are allowed

                   Year As Integer)

        Me.Day = Day

        Me.Month = Month

        Me.Year = Year

    End Sub


    Public Overrides Function ToString() As String      ' it can implement subroutines and functions

        Return String.Format($"{Year}/{Month}/{Day}")

    End Function


End Structure



Using structures


Dim date1 As MyDate

Dim date2 As New MyDate

Dim date3 As New MyDate With {.Year = 2000, .Month = 1, .Day = 1}

Dim date4 As New MyDate(2000, 1, 1)



CLS compliance requirements


  • equality and inequality operators are overloaded
  • Equals and GetHashCode must be overridden
