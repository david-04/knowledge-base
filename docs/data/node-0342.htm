Define a template


/**

* comment is required

*/

{template .sayHello private='true'          // can be called from everywhere in the current bundles (enables inlining)

                    autoescape='strict'}    // escape everything

    Hello world!

{/template}



Declaring parameters


/**

* Greet a person.

* @param name The name of the person       // parameters can be declared in the JavaDoc comment block

* @param? greeting Greeting (optional)

*/

{template .sayHello}

    {if not $greeting}

        Hello {$name}!

    {else}

        {$greeting} {$name}!

    {/if}

{/template}


/**

* Greet a person.

*/

{template .sayHello}

    {@param  name:     string}              // parameters can also be declared with commands (must be on top of the template)

    {@param? greeting: string}

    ...

{/template}



Calling templates


{alias my.package.general}                  // like import: can now call templates via my.package.general.myTemplate

{alias my.package.general as myLib}         // same as above but use a custom alias


{call .myTemplate}                          // prepend the namespace (or alias) to call templates from elsewhere

    {param show: $isActive /}               // pass parameters individually

    {param message}Hello world!{/param}

{/call}


{call .myTemplate data="$myMap" /}          // pass parameters from a map (name=key, value=value)


{call .myTemplate data="all" /}             // pass all parameters of the current template

                       

{call .myTemplate data="all" }              // pass all parameters from the current template

    {param message}Hello world!{/param}     // ... plus additional ones

{/call}
