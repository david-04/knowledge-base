Command line parameters


gawk [program]   -f --file program.awk          # read from file

                 -e --source "inline_code"      # program code passed as command line parameter

                 -i --include library.awk       # load the AWK library from AWKPATH (same as @include)


     [options]   -M --bignum                    # enable arbitrary precision arithmetic (not supported)

                 -O --optimize                  # performs constant folding only

                 -S --sandbox                   # disable system() and I/O redirection to files

                 -b --characters-as-bytes       # treat every byte as a single token (no UTF decoding)


     [analysis]  -L --lint fatal                # optional parameter "fatal": warnings abort the program

                 -p --profile=awkprof.out       # run profiling (execution count only, no measurement of duration)

                 -d --dump-variables=file       # create a dump file with


     [variables] -F --field-separator ","       # use the specified field separator for input splitting

                 -v --assign variable=value     # assign a value to a user-defined variable


                 --                             # end of command line options (stop argument parsing)


     [input]     file-1.txt                    

                 variable=value                 # NOT accessible in BEGIN (only in the rules section)

                 file-2.txt

                 -                              # stdin (cat file | sed -f program.awk file1.txt - file2.txt)



Environment variables


AWKPATH                                         # search path for *.awk source files

AWKLIBPATH                                      # search path for shared libraries (dll)

GAWK_READ_TIMEOUT                               # milliseconds timeout for read operations

GAWK_SOCK_RETRIES                               # retry count for TCP/IP socket operations

AWK_HASH=gst                                    # use a slightly faster hashing algorithm

GAWK_MSG_SRC                                    # show file and line number where an error occurred
