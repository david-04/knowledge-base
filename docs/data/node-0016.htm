> mongofiles put myfile.txt                         // upload and store the given (local) file


> mongofiles list                                   // list all documents


> mongofiles search my                              // list all documents that start with my*


> mongofiles get myfile.txt                         // download the file with the given name


> mongofiles get_id 585dd8a8146bde0dd0aeb299        // download the file with the given "_id" field


> mongofiles delete my-file.txt                     // delete all files with the given name


> mongofiles delete_id 585dd8a8146bde0dd0aeb299     // delete the file with the given "_id" column
