Absolute paths


/                                                   // the document root (the root element + comments + processing instructions)

/myRootElement/myChildElement                       // a path leading to the selected element

/myRootElement/*/myGrandChildElement                // path with one wildcard segment


//myElement                                         // myElement located anywhere within the whole document (can be the root element as well)

//@myAttribute                                      // myAttribute of any element anyhwere within the whole document



Relative steps


                                                    // relative steps match only direct children of the current context node


.                                                   // the current context node

..                                                  // the current context node's parent



myElement                                           // an element with the given name as a direct child of the

*                                                   // any element (but not attributes, text, comments and processing instructions)

node()                                              // any element, attribute, text, comment or processing instruction

myns:*                                              // any element in the given namespace


@myAttribute                                        // an attribute of the current context node

@*                                                  // all attributes of the current context node

@myns:*                                             // any attribute in the specified namespace



text()                                              // text content (PCDATA) of the current context node (usage: myChildElement/text())

comment()                                           // an XML comment as a direct child of the current context node

processing-instruction()                            // all XML processing instrcuctions (<?...?>) within the current context node

processing-instruction("xml-stylesheet")            // selected XML processing instructions within the current context node


myChild/@myChildsAttribute                          // realtive path

person/phone/../name                                // going back up: get the names of all persons who have a <phone> attached

.//@myAttribute                                     // myAttribute in the current context node or any direct or indirect child node



Unions


myElement|@myAttribute|/myRoot                      // match any of the specified paths or steps



Conditions


person[profession="programmer"]/name                // get the names of all programmers

people/profession[.="programmer"]["age &gt; 40"]    // multiple predicates can be combined


[.="programmer"]

[@myAttribute="myValue"]

[age &gt;= 18]                                      // < <= = => > (the less-than symbol has to be masked in the XSLT file)

[city != "Melbourne"]



[age=90 or age=100]                                 // and / or

[myStringElement]                                   // the element myStringElement is present and not empty



Unabbreviated paths


people/person/@id                                   // abbreviated form

child::people/child::person/attribute::id           // unabbreviated form


axis::target[predicate]

+-- ancestor                                        // all direct and indirect parent nodes

+-- ancestor-or-self                                // all direct and indirect parent nodes or the context node itself

+-- descendant                                      // all direct an dindirect children

+-- following-sibling                               // all siblings (brother/sister nodes) after the current one

+-- preceding-sibling                               // all siblings (brother/sister nodes) before the current one

+-- following                                       // all nodes after the current one

+-- preceding                                       // all nodes before the current one

+-- namespace                                       // all applied namespaces (even if defined in a parent node)

+-- self                                            // the context node itself (usage: self::*)
