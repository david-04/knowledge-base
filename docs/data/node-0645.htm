Automatic properties


Public Property Age As Integer                  ' read/write auto-implemented property


Public ReadOnly Property IsOfAge As Boolean     ' read-only auto-implemented property (can be assigned only once)



Properties backed by fields


Private _name As String                         ' source field

Public Property Name As String                  ' read/write property

    Get

        Return _name

    End Get

    Set(value As String)

        _name = value

    End Set

End Property


Private _country As String

Public ReadOnly Property Country As String      ' read-only property

    Get

        Return _country

    End Get

End Property


Private _pin As String

Public WriteOnly Property Pin As String         ' write-only auto-implemented property

    Set(ByVal value As String)

        _pin = value

    End Set

End Property



Default properties


Private data() As Integer = {1, 2, 10, 20}

Default Public Property _

    GetData(ByVal index As Integer) As Integer  ' passing parameters is possible for all properties (not only the default one)

    Get

        Return data(index)

    End Get

    Set(ByVal value As Integer)

        data(index) = value

    End Set

End Property
