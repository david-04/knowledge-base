Publishing the master


> git push origin master                            // upload changes made to the master branch to the origin repository

> git fetch origin                                  // download the latest commits (for all branches) from the origin repository

> git log --no-merges myBranch..origin/master       // show all commits in origin/master that are not in the local master branch

> git merge origin/master                           // merge the latest master commits into the local (master) branch

...

> git push origin master                            // upload changes made to the master branch to the origin repository



Consolidating branches


> git fetch origin                                  // download the latest commits

> git merge origin/remoteBranch                     // merge a remote branch into the currently checked out one

> git push -u origin myBranch:remoteBranch          // upload the current branch but name rename it to the original "remoteBranch"



Rebasing before push


git checkout myBranch                               // select the branch to rebase

git rebase origin/master                            // re-create the current branch on top of origin/master

git push -f origin myBranch                         // upload the rebased branch (must use -f because the commit history of

                                                    // myBranch has been altered; instead of rebasing it would therefore be

                                                    // better to create [and merge into] another branch "myBranch-v2")
