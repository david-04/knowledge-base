Configuration files


/etc/gitconfig                                      // global configuration (on Windows: C:\ProgramData\Git\config)

~/.gitconfig                                        // system-wide user configuration (on Windows: C:\Users\$USER\.gitconfig)

./.git/config                                       // repository-specific configuration



Maintaining configuration


> git config --list                                 // show all settings


> git config {--global|--system|Ø} key value        // save a setting globally, system-wide per user or project-specific (Ø)



Properties


user.name           "David Hofmann"                 // name and email address must be set (are used to document commits)

user.email          david@home.com


core.editor         notepad                         // the editor to use for capturing commit messages

merge.conflictstyle diff3                           // use diff3 to represent merge conflicts (will show not only the

                                                    // left and right side, but also the common ancestor)


alias.co            checkout                        // create a command alias ("git co" will be executed as "git checkout")

alias.unstage       "reset HEAD --"                 // aliases can consist of more than one word/parameter

alias.dir           "!ls -l"                        // "git dir" will execute the shell command "ls -l"



File filters


# comment                                           // filename: .gitignore

*.log                                               // wildcards are supported like in the shell (*, ? [one character] and [0-9])

!summary.log                                        // include this file, even if it is matched by an exclusion mask

generated/                                          // exclude a whole folder

folder/**/subfolder                                 // ** matches any sequence of subfolders

/*.log                                              // rule applies to this folder only (otherwise it's recursive by default)


git update-index --assume-unchanged myfile.txt      // hide the given file from git status/diff
