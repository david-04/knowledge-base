Conditions


<span ng-if="mustShowThis">...</span>               <!-- remove the element if the condition evaluates to false     -->

<span ng-hide="!mustShowThis">...</span>            <!-- set display:none if the condition evaluates to true        -->

<span ng-show="mustShowThis">...</span>             <!-- set display:none if the condition evalueates to false      -->



Switch-case


<div       ng-switch on   = "representation">       <!-- show only elements matching a given attribute              -->

    <table ng-switch-when = "Table"> ... </table>

    <ul    ng-switch-when = "List">  ... </ul>

    <p     ng-switch-default>        ... </p>

</div>



Iterating collections


<p ng-repeat="user in users | filter:{type:'A'}"    <!-- repeat this element for each element in the users array    -->

   ng-class-{even|odd}="myCssClass">                <!-- apply this CSS class for even/odd iterations only          -->

    {{ user.name }}                                 <!-- access the current element via the "user" variable         -->

    {{ $index    }}                                 <!-- contains the current loop index (starting at zero)         -->

    {{ $first    }}                                 <!-- boolean indicating if $index==1                            -->

    {{ $middle   }}                                 <!-- boolean indicating if $index is in ]1;MAX[                 -->

    {{ $last     }}                                 <!-- boolean indicating if $index==MAX                          -->

    {{ $even     }}                                 <!-- boolean indicating if $index is even                       -->

    {{ $odd      }}                                 <!-- boolean indicating if $index is odd                        -->

</p>


<p ng-repeat-start="user in users">                 <!-- repeat all sibling until (including) ng-repeat-end         -->

    Name: {{user.name}}

</p>

<p ng-repeat-end>                                   <!-- ng-repeat-end marks the last sibling to include            -->

    Age: {{user.age}}                               <!-- siblings can access loop-data as well                      -->

</p>



Iterating object properties


<p ng-repeat="property in object">{{property}}</p>  <!-- iterate over object property values (not property names!)  -->

<p ng-repeat="(key,value) in object">...</p>        <!-- retrieve each object property as a key-value-pair          -->
