Iteration


<xsl:for-each select     = "library/book">          <!-- for (select is an XPath that returns a set of nodes              -->


  <xsl:sort   select     = "author"                 <!-- optionally: sort by a given (XPath) child node content           -->

              data-type  = "{text|number}"

              lang       = "en-au"

              case-order = "{lower|upper}-first"

              order      = "{ascending|descending}"/>


  Title: <xsl:value-of select="title"/>             <!-- the template content is executed in the current book's context   -->


</xsl:for-each>



Conditional processing


<xsl:if test="price &lt; 10">                       <!-- if  (test is an XPath that evaluates to a boolean value)         -->

  really cheap

</xsl:if>



Switch-case


<xsl:choose>                                        <!-- switch case                                                      -->


  <xsl:when test="price &lt; 10">

    really cheap

  </xsl:when>


  <xsl:when test="price &lt; 20">

    affordable

  </xsl:when>


  <xsl:otherwise>

    quite expensive

  </xsl:otherwise>


</xsl:choose>
