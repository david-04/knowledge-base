Create and query the store


import { createStore } from 'redux';


var store = createStore(                            // create a new store instance (only one per application)

    myRootReducer,                                  // the reducer function (can be a combined chain)

    myInitialState                                  // the initial state to use (optional)

);


store.getState()                                    // retrieve the current state



Dispatch and listen to actions


store.dispatch(myAction)                            // dispatch an action to the reducer(s)


var unsubscribe = store.subscribe(() => {           // register an onChange event handler (returns an unsubscribe function)
