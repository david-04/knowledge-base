Layout managers


Canvas            ' fixed layout with absolute positioning

WrapPanel            ' re-flowing layout with automatic line wrapping

StackPanel            ' show elements in rows (above one another) or in columns (side-by-side)

Grid                    ' a table with cells organised in rows and columns

DockPanel            ' position content at the top, left, right or bottom

ViewBox         ' scale the whole content to fit into (and completeley fill) the client are


ScrollViewer    ' used as a wrapper for any of the above layout managers
