System views


sys.objects                                         -- contains all user-defined objects (column name is the name of the table/view/...)

sys.system_objects                                  -- contains the system-owned objects

sys.all_objects                                     -- contains both user-defined and system-owned object

sys.columns                                         -- contains all table columns

sys.database_principals                             -- contains all users, groups and roles


sys.indexes                                         -- contains all indices

sys.index_columns                                   -- contains all columns contained in indices

sys.hash_indexes                                    -- contains all hash indices


sys.dm_db_index_usage_stats                         -- contains index operations, their execution count and when they were last performed

sys.dm_db_missing_index_details                     -- contains detailed information about missing indices


sys.views                                           -- conains all views


information_schema.tables                           -- contains all tables in the current database that the current user can access

information_schema.columns                          -- contains all columns in the current database that the current user can access


sys.sql_modules                                     -- stored procedures



Help procedures


EXECUTE sp_help 'my_table_or_procedure';            -- retrieve information about any database object
