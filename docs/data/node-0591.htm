Assertions


assert(0 < x)                               // assertions can be enabled/disabled via -ea/-da JVM switches

assert(0 < x, "x must be positive")         // can pass a message to be included in the AssertionError


require(0 < x)                              // throws an IllegalArgumentException

require(0 < x, "x must be positive")        // can pass a message


val a = b / 3 ensuring(0 < _)               // ensuring asserts the value to its left-hand side
