Default databases


master                                              -- contains the metadata of all databases (should be backed up regularly)

model                                               -- template database that is used as copy-template when creating a new database (contains some system tables)

tempdb                                              -- used to store temporary tables and intermediate results

msdb                                                -- contains task scheduling, alerts and exception handling (SQL Server Agent)



Command line tools


> bcp david.dbo.customer out "customer.txt" -T -c   --Bulk copy: Import or export tables from/to flat text files

> sqlcmd                                            -- run SQL commands from the command line (or input files)

> sqlservr                                          -- start a database server instance



Change tracking


EXECUTE sys.sp_cdc_enable_db;                       -- enable change tracking for the whole database


EXECUTE sys.sp_cdc_enable_table

    @source_schema = 'my_schema',

    @source_name = 'my_table',

    @role_name = 'cdc_admin'



Backup and recovery


BACKUP DATABASE my_database

    TO 'C:\backup.bak' WITH COMPRESSION;



Normalisation


1st normal form => eliminate multivalued and composite attributes

2nd normal form => each non-key field can be derived only, if ALL key fields are taken into account

3rd normal form => no no-key field can be derived from another non-hey field
