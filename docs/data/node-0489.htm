Nested rules


#myElement {


    color: black;


    .myClass {                                      // create a rule for selector "#myElement .myClass" ("&" refers to all

        width: 300px;                               // parent selectors in the chain, in case of multi-level nesting)

    }


    &:hover {                                       // use "&" to reference the parent selector

        color: red;

    }


    &:active, &+&, &Parent > &.myClass {            // the "&" can be used repeatedly and in variations

        height: 3em;

    }

}



Guards


.myClass when (@myVariable = myValue) {             // output this rule only if the given condition is met

    color: red;

}



Property merging


.myClass {


    box-shadow+: inset 0 0 10px #555;               // combine all values into a comma-separated list

    box-shadow+: 0 0 20px black;


    border+_   : 1px;                               // combine all values into a space separated list

    border+_   : solid;

    border+_   : black;

}
