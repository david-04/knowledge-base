Defining an XML data source


<asp:XmlDataSource ... DataFile      = "~/MyData.xml"      <!-- pass a local file or a link to an online resource         -->

                       Data          = "<?xml ..."         <!-- hard-coded XML content                                    -->

                       XPath         = "books/book"        <!-- retrieve nodes by XPath and return the nodes' attributes  -->

                       TransformFile = "~/Transform.xslt"  <!-- apply an XSLT transformation                              -->

                       Transform     = "inline xslt" />    <!-- apply a hard-coded transformmation                        -->



Accessing the underlying XML data


this.XmlDataSource.GetXmlDocument();                       // retrieve the XML document

this.XmlDataSource.Save();                                 // save changes back to the XML file
