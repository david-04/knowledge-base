Standalone tags


@param                                              // function parameter

@return                                             // return value

@exception                                          // exceptions thrown

@throws                                             // exceptions thrown


@author                                             // name of the author

@since       v2.1                                   // when the element was introduced

@see         myPackage.myClass#myProp Label         // cross-link

@version                                            // the version of a class


@deprecated                                         // deprecated item


@serial                                             // default serializable field

@serialData                                         // data written by writeObject() or writeExternal()

@serialField                                        // ObjectStreamField component



Inline tags


{@code       System.out.println("...");     }       // display verbatim as source code

{@literal    42                             }       // display verbatim as a literal

{@value      myPackage.myClass#myProp       }       // display as a static field constant


{@docRoot                                   }       // path to the root directory of the current documentation

{@inheritDoc                                }       // inherit a comment from the immediate superclass

{@link       myPackage.myClass#myProp Label }       // in-line link to another topic

{@linkplain  myPackage.myClass#myProp Label }       // in-line link to another topic, but the link is displayed in a plaintext font
